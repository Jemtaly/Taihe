#+TITLE: 在项目的 gn 构建流程中加入 Taihe

* 如何将 Taihe 引入项目：

需要将 Taihe 中的 *runtime* 和 *compiler* 部分加入到项目构建流程。

- Taihe runtime 部分产物为 libtaihe-rt.so，可以通过设置 libs 和 lib_dirs 字段来设置依赖

  #+BEGIN_SRC gn
  shared_library("author") {
    lib_dirs = [ libtaihe_rt_path ]
    libs = ["taihe-rt"]
  }
  #+END_SRC

- Taihe compiler 部分的运行调用使用 taihe_compiler.gni 文件进行了封装，在使用时直接在 BUILD.gn 中使用 taihe_compiler 模板实例化，并设置到 deps 中即可

  1. *使用 taihe_compiler 模板：* 实例化一个 taihe_compiler，指定需要编译的 .taihe 文件，以及编译的模式

     #+BEGIN_SRC gn
     taihe_compiler("author_compiler") {
       sources = [
         "xxx.taihe"
       ]
       mode = "author"
     }
     #+END_SRC

     - sources： 需要编译的 .taihe 文件，taihe_compiler 会生成对应的头文件

     - mode： taihe_compiler 提供两种编译模式，author 模式和 user 模式 

  2. *设置依赖：* 通过 deps 字段引入 "author_compiler"，会帮用户基于提供的 .taihe 文件生成头文件，并将这些头文件引入到构建过程中

     #+BEGIN_SRC gn
     shared_library("author") {
       deps = [
         ":author_compiler,
       ]
     }
     #+END_SRC

* 示例项目：

*路径：* TaiheCompiler/test/integer

*步骤：*

1. clone taihe 项目并编译出 libtaihe-rt.so

   #+BEGIN_SRC gn
   git clone git@github.com:Jemtaly/TaiheCompiler.git
   cd TaiheCompiler
   mkdir -p out/runtime
   cd out/runtime
   cmake -DCMAKE_C_COMPILER={clang_path} -DCMAKE_CXX_COMPILER={clang++_path} ../../runtime && make
   #+END_SRC

2. 将 taihe_compiler.gni 复制到项目的 build 目录下

   *taihe_compiler.gni  的存放路径：* TaiheCompiler/build/compiler/taihe_compiler.gni，在示例项目 test/integer 中可以将模板移动到 build/compiler 下

3. 在 BUILDCONFIG.gn 中指定 【taihe 源代码路径】和 【libtaihe-rt.so 路径】

   #+BEGIN_SRC gn
   taihe_source_path = "/root/TaiheCompiler"
   libtaihe_rt_path = "$taihe_source_path/out/runtime"
   #+END_SRC

4. 将 taihe 加入到 author 和 user 的 BUILD.gn（以 author 的为例）

   - 使用 taihe_compiler：

     #+BEGIN_SRC gn
     import("//build/compiler/taihe_compiler.gni")
     
     # 实例化 taihe_compiler，指定编译模式为 "author"
     taihe_compiler("integer_test_author_compiler") {
       sources = [
         "../idl/integer.arithmetic.taihe",
         "../idl/integer.io.taihe"
       ]
       mode = "author"
     }
     
     # 设置依赖
     shared_library("author") {
       deps = [
         ":integer_test_author_compiler",
       ]
     }
     #+END_SRC

   - 设置依赖 libtaihe_rt.so：

     #+BEGIN_SRC gn
     shared_library("author") {
       lib_dirs = [ libtaihe_rt_path ]
       libs = ["taihe-rt"]
     }
     #+END_SRC
